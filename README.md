SkyBooker is a web-based airline reservation system built using
Python (Flask) and SQLite.

## Features
- User Registration & Login
- Flight Search
- Ticket Booking
- E-ticket Generation

## Technologies Used
- Python
- Flask
- SQLite
- HTML & CSS

## How to Run
1. Install dependencies:
   pip install flask
2. Initialize database:
   python database.py
3. Run application:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
